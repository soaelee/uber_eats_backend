# Uber Eats

The Backend of Uber Eats Clone

## User Model:

- id
- createdAt
- updatedAt

- email
- password
- role(client|owner|delivery)

## User CRUD:

- Create Account (create => save)
- Log In (findById)
- See Profile (find?)
- Edit Profile (update)
- Verify Email (find?)
